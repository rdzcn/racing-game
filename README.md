# Racing

Browser-based 3D car racing game for kids. Vite + React + TypeScript,
@react-three/fiber + rapier physics. See `PLAN.md` for the roadmap.

```sh
pnpm install
pnpm dev        # play at localhost:5173
pnpm test       # unit tests
pnpm lint && pnpm typecheck
```

Controls: arrow keys or WASD.
